# Gorilla Godot
## By baggZ, assisted by KinexDev

### Why Gorilla Godot?
Gorilla Godot uses [Gorilla Locomotion Godot](https://github.com/kinexdev/GorillaLocomotion-Godot) as the base of Gorilla Godots locomotion. However we have upgraded the code with more features.

### Features
- Gorilla locomotive systems
- Hand feedback(sounds and haptics)
- A starter player model
- Rigging scripts
- A testing map
